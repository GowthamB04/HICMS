Frontend quick reference
========================

This folder contains short notes for each main Angular component to help onboarding and maintenance.

- `dashboard.txt` — Overview of dashboard responsibilities and data sources.
- `claims.txt` — Claims page responsibilities and API calls.
- `policies.txt` — Policies page responsibilities and API calls.
- `payments.txt` — Payments page responsibilities and API calls.
- `documents.txt` — Documents upload/requirements (four docs per claim).
- `users.txt` — User management and profile notes.
- `analytics.txt` — Admin analytics and where to find totalPaidAmount.
- `login.txt` — Login flow and auth service notes.
